# EEGLAB interoperability

The pages below describe how EEGLAB compares to and interacts with other software and environments.

- [**EEGLAB and FieldTrip**](/others/EEGLAB_and_FieldTrip.html) — use EEGLAB together with the FieldTrip toolbox for source analysis and statistics.
- [**EEGLAB and High Performance Computing**](/others/EEGLAB_and_high_performance_computing.html) — run EEGLAB on clusters and supercomputers for large-scale processing.
- [**EEGLAB and Python**](/others/EEGLAB_and_python.html) — use EEGLAB alongside Python tools (e.g. MNE) and call EEGLAB from Python.
- [**EEGLAB on Octave**](/others/Running_EEGLAB_on_Octave.html) — run EEGLAB under GNU Octave, a free alternative to MATLAB.
- [**EEGLAB vs. commercial software**](/others/EEGLAB_vs_Commercial_EEG_Software.html) — how EEGLAB compares to commercial EEG analysis packages.
